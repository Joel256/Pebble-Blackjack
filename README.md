# Pebble-Blackjack
Traditional Blackjack for Pebble watches. Player vs Dealer, Single deck. Fast paced. Game loads instantly, you can play a head in just a few seconds. 
Designed for Pebble 2 Duo and Pebble Time 2. 

The game is available on all Pebble watches except Round and Round Time 2.
