# Pebble-Blackjack
Traditional Blackjack for Pebble watches. Player vs Dealer, Single deck. Fast paced. 
Designed for Pebble 2 Duo. Not available to round watches at this time.

Available on:
Pebble / Aplite,
Pebble Time / Basalt,
Pebble 2 / Diorite,
Pebble 2 Duo / Flint.

When I get a Pebble Time 2 and Pebble Round 2, I will investigate adding color and porting it to those devices.

Made by: Persistent Productions which is Joel Penton.
